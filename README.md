# Sales-Distribution-Performance-Dashboard
## 📊 Sales & Distribution Performance Dashboard

### 🏷️ Dashboard Overview

The **Sales & Distribution Performance Dashboard** is an interactive and user-centric report designed to analyze sales, distribution, and inventory performance across regions and distributors.

This dashboard incorporates **Row-Level Security (RLS)**, ensuring that users can only view data relevant to their assigned roles, making it suitable for real-world business environments with controlled data access.

---

### 🔄 Key Features

* 🔐 **User-Level Access (RLS):**
  Implemented Row-Level Security to restrict data visibility based on user roles, enhancing data privacy and security.

* 🔁 **Dynamic Metric Switching (Value vs Quantity):**
  Integrated a toggle feature allowing users to switch between **Value** and **Quantity** views.

  * Selecting *Value* displays revenue-based insights
  * Selecting *Quantity* reflects volume-based analysis
    This enables flexible analysis based on business needs.

---

### 📈 Dashboard Insights & Visuals

* **KPI Cards (Top Section):**
  Displays key metrics such as Total Distributors, Total Sales, and Total Purchase, providing a quick summary of overall performance.

* **State-wise Sales Map:**
  A geographical visualization highlighting sales distribution across different states, helping identify high-performing regions.

* **Data Upload Status (Donut Chart):**
  Compares current vs previous month data uploads, enabling monitoring of data completeness and reporting consistency.

* **Monthly Sales Trend (Line Chart):**
  Tracks Primary and Secondary sales over time, helping identify trends, seasonality, and performance fluctuations.

* **Monthly Inventory Trend:**
  Shows inventory movement across months, assisting in stock planning and demand forecasting.

* **Sub-Region & Division-wise Sales (Bar Chart):**
  Breaks down sales by sub-regions and divisions, providing insights into regional performance and business segmentation.

* **Top 10 Distributors (Bar Chart):**
  Highlights top-performing distributors based on selected metric (Value/Quantity), enabling quick identification of key contributors.

### 🛠️ Tech Stack

* **Power BI Desktop** – Dashboard development and visualization
* **DAX (Data Analysis Expressions)** – Dynamic measures and calculations
* **SQL Server Management Studio (SSMS)** – Extracted and transformed data using stored procedures for efficient integration into Power BI
* **Row-Level Security (RLS)** – User-based data access control
